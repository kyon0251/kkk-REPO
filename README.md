

# UG Control System (水下滑翔机控制系统)

这是一个基于 Arduino 架构开发的姿态与运动控制系统代码库。该项目通过整合 H30 惯性测量单元（IMU）、带有编码器的推杆电机（Push Rod）和基于串口通信的控制电机（Motor），实现了载具的俯仰角（Pitch）PID 闭环控制、重力/浮力驱动下的下潜与上浮控制，以及航向转向控制。

## 🌟 核心特性

* **多模式运动调度**：顶层主控类 `UG` 封装了 `movePosMode`（定点位移模式）、`movePIDMode`（PID 姿态追踪模式）和 `turnMode`（定时转向模式）等高阶控制接口，使得主逻辑极为简洁。
* **串级 PID 俯仰控制**：`PitchControl` 模块采用串级 PID 算法，以外环角度误差和内环角速度误差计算输出，动态调节推杆以维持目标俯仰姿态。
* **位移闭环反馈**：推杆模块 (`PushRod`) 利用硬件外部中断实时读取编码器脉冲，实现毫米级的精准位移与限位控制（最大行程设计限制为 21800 个脉冲，约合 122mm）。
* **高效传感器解析**：`H30_IMU` 模块内置状态机，可高效解析 IMU 串口协议帧，实时更新加速度、角速度、欧拉角（Pitch/Roll/Yaw）和四元数。

## 🗂️ 代码结构与模块说明

项目按面向对象思想进行了模块化划分，主要包含以下部分：

* **`UG`**：系统顶层调度器，负责初始化各子模块并在不同运行模式（如上下潜循环、姿态修正、电机转向）之间进行协调与切换。
* **`H30_IMU`**：负责监听指定串口（默认为 `Serial1`），校验与解析 H30 传感器数据包并提供统一的数据读取接口。
* **`PushRod`**：推杆电机驱动层。负责控制 L298N/TB6612 等电机驱动模块的 PWM 输出及正反转引脚（IN1/IN2），并通过中断引脚（Pin 2）累加编码器计数。
* **`PitchControl`**：俯仰角控制器。读取 IMU 反馈的当前 Pitch 和 Y 轴角速度，进行 PID 计算后向 `PushRod` 发送死区控制指令。
* **`Motor`**：旋转电机控制协议层。通过串口（默认为 `Serial2`）下发特定功能码与控制帧，支持控制电机的速度模式、位置模式、回零等复杂指令。

## 🛠️ 硬件与引脚配置

根据代码中的定义，您的 Arduino 硬件连线需参考以下配置：

**推杆电机引脚 (Push Rod)**
* `PR_ENCODER`: 引脚 `2` (必须支持外部硬件中断)
* `PR_DIRECTION`: 引脚 `3` 
* `PR_IN2`: 引脚 `4`
* `PR_IN1`: 引脚 `5`
* `PR_PWMA`: 引脚 `6` (用于 PWM 调速)
* `PR_STBY`: 引脚 `7`

**串口通信配置**
* `Serial`: 115200 波特率 (用于电脑端调试打印)
* `Serial1` (IMU_SERIAL): 115200 波特率 (连接 H30 IMU 模块)
* `Serial2` (MOTOR_SERIAL): 115200 波特率 (连接转向控制电机)

## 🚀 快速运行

1. 下载并安装 [Arduino IDE](https://www.arduino.cc/en/software)。
2. 确保你的目标开发板（如 Arduino Mega 2560，需具备 `Serial1` 和 `Serial2` 硬件串口）已连接。
3. 打开 `UG.ino` 项目文件。
4. 默认的主循环测试代码如下：
    ```cpp
    void setup() {
        ug.init(); // 初始化串口并自动进行推杆回零与电机设置
    }

    void loop() {
        // 每 10000ms 在 0mm 和 122mm 之间切换推杆位置
        ug.movePosMode(0, 122, 10000); 
        
        // 执行转向测试
        ug.turnMode(45, 5000, 5000, 2); 
    }
    ```
5. 编译并烧录代码至开发板。
